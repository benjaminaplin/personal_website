Real-time Political Fundraising
Node, Express, D3.js, JavaScript, HTML5, CSS3

While working in communications and marketing at international non-profits, I realized the power of data visualization. As a result I was excited to work with D3.js, a javascript data visualization library. This site pulls real-time data from the Federal Election Commission's API and the Sunlight Foundation's API, rendering the realtime fundraising results using D3.js, Node, Express and Foundation CSS.

Eventr - Live Music Nearby, Now
Ruby, Rails, Postgres, JavaScript, HTML5, CSS3

I worked in a team of four to produce an app we called Eventr, an application showing a dynamic map of concerts nearby the user that are happening that day. Collaborating using Git and Github, we built a Ruby on Rails application that enabled users to view music events around them using geolocation, see details on individual events, and save them to their profile. APIs including Google Maps and Spotify provided up-to-the-minute content for the site.
